# Implemented AI Agents

This document lists the AI agents implemented in this task and a short description of what each one does.

## Tier 1 Agents

| Slug | Functionality (Short) |
|---|---|
| `deal-ai-chat` | Conversational deal copilot for strategy, risk detection, and next actions. |
| `deal-daily-briefing` | Daily pipeline summary with key changes, stale deals, and focus actions. |
| `quick-deal-email` | Fast, context-aware follow-up email drafting for active deals. |
| `lovable-prototype-builder` | Converts requirements into prototype-ready app specs and prompts. |
| `client-call-analyzer` | Analyzes call transcripts for sentiment, risks, opportunities, and follow-ups. |
| `client-communication-coach` | Coaches messaging tone, clarity, and stakeholder communication quality. |
| `meeting-efficiency-analyzer` | Scores meeting efficiency and suggests measurable improvement actions. |
| `eos-pattern-detective` | Detects recurring EOS issue patterns and likely root causes. |
| `eos-pod-health` | Assesses pod health from EOS/accountability execution signals. |
| `eos-quarterly-digest` | Generates a quarterly EOS digest with wins, risks, and priorities. |
| `bug-feature-planner` | Turns bugs/features into actionable implementation and rollout plans. |
| `code-review-generator` | Produces structured, risk-focused code review feedback/checklists. |
| `technical-plan-generator` | Builds end-to-end technical plans from product requirements. |
| `project-analyzer` | Evaluates project health, timeline risk, and execution bottlenecks. |

## Tier 2 Agents

| Slug | Functionality (Short) |
|---|---|
| `email-draft-generator` | Server-side draft generation using deal, client, and contact context. |
| `sow-generator` | Generates statement-of-work content with export-ready PDF output. |
| `meeting-intelligence` | Single-pass structured extraction of summary, actions, issues, and risks. |
| `smart-meeting-categorizer` | Categorizes meetings with confidence, rationale, and topic tags. |
| `meeting-issue-reporter` | Extracts blockers, concerns, and risks from meeting discussions. |
| `accountability-overlap-analyzer` | Detects overlapping ownership and ambiguous role responsibilities. |
| `accountability-chart-reminder` | Sends reminder prompts to review accountability chart ownership. |
| `accountability-manager-nudge` | Sends manager nudges to resolve accountability clarity gaps. |
| `accountability-revisit-reminder` | Sends recurring reminders to revisit role boundaries and ownership. |
| `hr-request-processing` | Triage HR requests into structured tasks with urgency and next steps. |
