## Summary

- **Prompt Template Management** page under Admin → KNOWLEDGE & AI → AI Hub → Prompt Templates.
- Full CRUD for reusable AI prompt templates with placeholders (e.g. `{{recipient_name}}`, `{{topic}}`).
- UI matches the design: breadcrumbs, header with “Back to Dashboard” and “+ Create Template”, four summary stat cards (Total Templates, Active Templates, Total Usage, Categories), and a list of templates with preview, Edit/Duplicate/Delete.
- **Create / Edit** modal with: Template Name, Slug, Description, Category (dropdown), Template Content (required). Subtitle: “Design a reusable prompt template for AI agents.”

## Changes

- **Database:** New table `prompt_templates` (migration `20260218000000_prompt_templates.sql`) with `name`, `slug`, `description`, `category`, `template_content`, `is_active`, `usage_count`, RLS, and seed templates (Professional Email, Follow-up Email, Meeting Summary, Deal Update).
- **Cache:** `queryKeys.ai.promptTemplates` / `promptTemplate(id)` and `invalidateKeys.promptTemplates` in `src/lib/cache.ts`.
- **Hook:** `src/hooks/usePromptTemplates.ts` — `usePromptTemplates`, `usePromptTemplateStats`, `useCreatePromptTemplate`, `useUpdatePromptTemplate`, `useDeletePromptTemplate`, `useDuplicatePromptTemplate`.
- **Page:** `src/pages/admin/ai/PromptTemplateManagement.tsx` — list view, stat cards, Create/Edit dialogs, delete confirmation.
- **Routing:** Route `/admin/ai/prompt-templates` and nav link “Prompt Templates” → `/admin/ai/prompt-templates` (was `/admin/ai-models`).

## How to test

1. Run migrations: `npm run migrations:run`
2. Go to **Admin** → **KNOWLEDGE & AI** → **AI Hub** → **Prompt Templates** (or `/admin/ai/prompt-templates`)
3. Confirm summary stats and seeded templates (e.g. Professional Email, Follow-up Email).
4. Create a new template, edit one, duplicate one, delete one (with confirmation).
